hitchhike memories
============================

## Requirements
- hugo v0.54.0

## First
- `hugo`

```
brew install hugo
```


## Devlopment
- `http://localhost:1313`

```
hugo server -D -w
```


## Domain
https://hitchhike.work


## Usage
1. add photos to `others/hitchhike`
2. exec `new_post.py`

```
python new_post.py
```

3. Optionally adjust image rotation
