About

ひろたつ。1994年12月8日生まれ。東京出身。学生時代に初めてヒッチハイクを経験。ヒッチハイクの魅力にハマり、九州から北海道まで旅をする。大学卒業後都内のIT企業に就職。退職後にヒッチハイクで日本一周を達成。その後ニュージーランドでのワーキングホリデーを開始。ヒッチハイクでニュージーランド一周中。

hitchhike memories

今までにヒッチハイクで乗せて下さった方々との思い出です。全ての方と写真を撮らせて頂いたわけでは無いため、実際に乗せて下さった方はもっといます。日本にいる間に100台以上、ニュージーランドでも50台ほどのヒッチハイクを経験しました。ヒッチハイクの魅力が伝わればと思い、このサイトを立ち上げました。
