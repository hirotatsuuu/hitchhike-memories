About

ひろたつです。1994年12月8日生まれの東京都杉並区出身です。プログラミングが好きな甘党です。学生時代に初めてヒッチハイクを経験しました。大学卒業後は都内のIT企業にエンジニアとして就職しました。会社員をしている間にも、長期休暇のたびにヒッチハイクで旅に出ていました。退職後にヒッチハイクで日本一周を完遂しました。現在はニュージーランドでワーキングホリデーをしながらヒッチハイクで旅をしています。そんなやつです。

hitchhike memories

今までにヒッチハイクで乗せて下った方々との思い出の写真たちです。日本で100台以上、ニュージーランドで50台ほどのヒッチハイクを経験しました。今までにヒッチハイクを通して関わった方々への感謝と、ヒッチハイクに興味がある人に向けてヒッチハイクの魅力が伝わればと思いこのサイトを立ち上げました。ヒッチハイクをするたびに随時更新します。足を運んで下さりありがとうございます。乗せて下さった方々にぼくの想いが届くといいな…
